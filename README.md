# weather-data-visualizer--Arth-Rana
Air Quality Analysis Project (Delhi)
This project analyzes daily air quality (AQI) data from Delhi using Python. The aim is to study pollution patterns over time by cleaning the dataset, calculating key statistics, creating visualizations, and identifying trends.

Project Includes:
	•	Loading real AQI data
	•	Cleaning and preprocessing
	•	Calculating averages and monthly trends
	•	Visualising AQI levels with Matplotlib
	•	Grouping data by month and season
	•	Exporting cleaned data

Python Libraries:
	•	Pandas – data loading and cleaning
	•	NumPy – numerical/statistical calculations
	•	Matplotlib – visualisation
